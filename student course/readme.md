# Student Course Management System

## 📌 Project Overview

The Student Course Management System is a web-based application designed to manage student and course-related information efficiently.

This project provides a simple and user-friendly interface for managing student records and course information.

## ✨ Features

- Add new students
- View student records
- Update student information
- Delete student records
- Add and manage courses
- Enroll students in courses
- View course information
- User-friendly interface
- Organized student and course management

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```text
student-course/
│
├── index.html

├── script.js
└── README.md
```


## 🚀 How to Run

1. Download or clone this repository.
2. Open the project folder in VS Code.
3. Open the `index.html` file in a web browser.

You can also use the Live Server extension in VS Code:

1. Open `index.html`.
2. Right-click on the file.
3. Select **Open with Live Server**.

## 🎯 Project Objectives

The main objectives of this project are:

- To manage student information efficiently.
- To manage course records.
- To practice web development concepts.
- To create a user-friendly student course management system.
- To improve HTML, CSS, and JavaScript skills.
